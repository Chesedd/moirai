# Moirai

Персональный планировщик дел в Telegram, питающийся от Claude Code Routines.

Пишешь в бот всё, что приходит в голову — задачи, события, выполнения,
занятые слоты, заметки — свободным текстом. По расписанию облачные
Claude Code Routines читают накопленное, строят план на день, недельный
обзор, приоритизацию и расписание. Бот отправляет результат обратно,
следит за таймером напоминаний и реагирует на команды.

## Компоненты

- **Бот** (`bot/`) — Python 3.12 + aiogram, упакован в Docker, работает на VPS.
- **Routines** (`docs/prompts/`) — промпты для Claude Code Routines.
- **Хранилище** — общая папка в Google Drive (бот ходит через service
  account, Claude — через родной Google Drive коннектор).
